This application allows you to upload a CSV file and will create student accounts for webserver and FTP access.

CSV files should be in the format of firstname,lastname,username,password. Headers should be included. Files with invalid formats will not process.

Example

firstname lastname username password
Tiffany Ford fordt Qwerty123
